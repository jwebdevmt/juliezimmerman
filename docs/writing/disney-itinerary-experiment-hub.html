Five AI Instances Wrote Disney Itineraries for the Same Person

Disney Experiment · Same traveler. Same parks. Different long-context AI perspectives.

Five different AI instances wrote Disney itineraries for the same person — here’s what happened.

The prompt looked simple from the outside: create Disney itineraries for an experienced solo visitor who cares about atmosphere, memory, pacing, and practical movement through the parks.

The outputs were not simply different versions of the same plan. Each instance emphasized a different kind of intelligence: emotional register, compression, stage architecture, continuity, failure modes, and synthesis.

That difference is the point of this section. The itineraries are readable as Disney content, but they also show how preserved long-context review roles can produce visibly different artifacts.
