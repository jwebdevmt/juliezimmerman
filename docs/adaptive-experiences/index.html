Disney Itineraries as Preserved Perspectives

Adaptive Experiences · Disney is the interface. Preserved perspective is the experiment.

On the surface, this section contains Disney itineraries. Underneath that, it is a small experiment in perspective continuity: what happens when several long-context AI instances receive the same traveler, the same constraints, and the same planning problem, but approach it through different stable roles?

Disney works well as the test domain because there is no single correct answer. Atmosphere, pacing, memory, exhaustion, weather, transportation, nostalgia, food, crowd tolerance, and emotional goals all matter. A plan can be technically efficient and still feel wrong.

That makes the differences visible. Each itinerary below reflects a different planning lens.
